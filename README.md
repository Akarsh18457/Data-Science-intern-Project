## Trader Performance vs Market Sentiment

This project analyzes how Bitcoin market sentiment (Fear/Greed)
impacts trader behavior and performance on Hyperliquid.

### Setup
pip install pandas numpy matplotlib seaborn scikit-learn

### Run
Open notebooks/Data Analysis.ipynb and run all cells.

### Outputs
Charts showing:
- PnL vs sentiment
- Leverage behavior
- Trade frequency
- Trader segments
